# Mercado Connecting 2026

## Tema WordPress personalizado para Mercado Connecting
**Partner Oficial Mercado Libre**

### Descripción
Tema WordPress personalizado con animaciones IA, diseño responsive y sección Prensa interactiva.

### Estructura
```
mercado-connecting-2026/
├── style.css              ← Header del tema + estilos principales
├── index.php              ← Template principal
├── header.php             ← Cabecera con meta tags y OG
├── footer.php             ← Cierre del documento
├── functions.php          ← Enqueue scripts, theme support, menus
├── page.php               ← Template: "Mercado Connecting Home"
├── readme.txt             ← Documentación
└── assets/
    ├── js/main.js         ← Animaciones e interactividad
    └── css/responsive.css ← Breakpoints responsivos
```

### Características
- ✅ Diseño 100% responsive (480px / 768px / 1024px / 1200px)
- ✅ Animación IA con partículas conectadas en sección Prensa
- ✅ Símbolos flotantes: 📺 TV · 📻 Radio · 🎬 Video · 📡 Streaming
- ✅ Animación palabra por palabra con glow dorado
- ✅ WhatsApp integrado: 5493517629492
- ✅ Cards de Prensa clickeables (Foro, Infonegocios, TikTok)
- ✅ Footer copyright 2026

### Instalación en WordPress
1. Ir a **Apariencia → Temas → Añadir nuevo → Subir tema**
2. Subir el archivo `.zip`
3. Activar el tema
4. Crear una página con template **"Mercado Connecting Home"**
5. Establecerla como página de inicio en **Ajustes → Lectura**

### Breakpoints Responsive
| Dispositivo | Ancho | Columnas Prensa |
|-------------|-------|-----------------|
| Mobile      | ≤ 480px | 1 columna |
| Tablet      | 481-768px | 2 columnas |
| Desktop     | 769-1200px | 3 columnas |
| Large       | ≥ 1201px | 4 columnas |
